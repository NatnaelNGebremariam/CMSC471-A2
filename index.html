Interactive Weather Visualization

About the Dataset

Our visualization is based on a weather dataset containing over 416,000 records from various weather stations across the United States. The dataset is provided by the NOAA Daily Global Historical Climatology Networkincludes important meteorological variables such as temperature (TMIN, TMAX, TAVG), wind speed (AWND, WSF5),precipitation (PRCP), and snowfall (SNOW, SNWD). The goal of this project is to explore relationships between these variables and allow users to interactively analyze weather patterns.

Our visualization is implemented using D3.js and presents a scatterplot. We explored alternative visualizations, such as line charts and bar graphs, but ultimately chose a scatterplot due to its ability to depict correlations between multiple variables simultaneously.

Development Process

Our Group had 3 members. Two of the group members were incharge of analyzing the data set and creating the scatterplot using D3 which took 10 hours. The other group member helped with debugging, doing the write up on the webpage, and recording the demo video which took 2 hours. We chose the NOAA weather data set because there is a lot of information that can be extracted. Before we started to code we know that for this particular data set we wanted the user to be able to filter through stations and see the weather in all or specific states. To design the scatter plot first we gathered possible options users can make in order to filter through the data such as: "Station", "State" and etc. Then we made sure the user has a slider for the months to allow the user to view very specific data. Our main challenges while developing the scatter plot was organizing the data we tried to represent all the weathers orginially by state, but there was a issue of having data overlapping as the same state would have repeated or slightly different data this alone took 5 hours to resolve. Some aspects that were rather straightforward to implement were the slider and dropdown menu because we were able to practice this feature during our labs. If we were to do this project again we would change the station dropdown by removing it, and allowing comparison of weathers between states because users would want to compare the weather between different states.

How to Use This Visualization

  • Station Dropdown: Select a station to filter available states and data.
  • State Dropdown: Select a state based on the chosen station.
  • Month Slider: Slide to view data for a specific month.
  • X & Y Axis Dropdowns: Choose variables for the scatter plot.
  • Bubble Chart: Visualizes temperature, precipitation, and wind speed.
  • Color Coding: Each weather station is assigned a unique color based on elevation.
  • Zoom Feature: You can zoom unto the bubbles in order to see clearly!
